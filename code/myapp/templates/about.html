ReligioNer

Compare anything against religious texts.

Why:

Religions are often compared against each other. This application is a way to show how they are very similar in many instances. There are also a lot of other texts ranging from books to song lyrics that have their sources rooted in religion. This application is a way to explore those relationships further and more analytically.

How:

The backend of this app is written in Python using a bag of words approach along with the cosine similarity as the metric for determining similarity between texts. The application is deployed using Flask. The graphing was all done using gephi.
